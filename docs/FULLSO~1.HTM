Hi,  here is a solution if found in this news group.  It works well
though it produces large files:

BTW: Isn't there any FAQ out there for postscrip/pdf related problems? 
I'm not an expert for both and don't take this as offense but I see the
same questions asked and answered here all the time.  

Uli

Create a "wrapper" file that begins with
---snip---
%!PS
/_begin_job_
{
        /tweak_save save def
        /tweak_dc countdictstack def
        /tweak_oc count 1 sub def
        userdict begin
}bind def

/_end_job_
{
        count tweak_oc sub{pop}repeat
        countdictstack tweak_dc sub{end}repeat
        tweak_save restore
}bind def
---snip---
and append your PS files to this bracketing the files each with
_begin_job_
at the beginninng and
_end_job_
at the end.


----
OR
----
% Now, add your jobs like this:
_begin_job
(oneofyour_ps_files)run
_end_job_
% and so on.
---snip---
where "oneofyour_ps_files" is the full path name of the PS file. If you
use '\' in your
path names, be sure to write them as '\\'.
