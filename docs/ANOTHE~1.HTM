> Does anyone have a way of merging ps-documents without buying expensive

Manually editing the ps file?  take `showpage' out of each ps file, then
try inserting the ps code in-between a gsave and grestore

%!
%% header stuff
gsave
%% first file
grestore
%% continue above sequence sans header stuff
showpage

see if that will work for you.
